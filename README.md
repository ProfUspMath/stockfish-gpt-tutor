# Stockfish GPT Chess Tutor

Un tutor de ajedrez interactivo que analiza partidas en formato PGN. Utiliza la potencia de cálculo de **Stockfish** (vía la API de Lichess Cloud Eval) y la capacidad explicativa de **OpenAI GPT** para redactar análisis didácticos y literarios de cada posición.

## Créditos y Atribuciones
Este proyecto es una implementación independiente inspirada en el concepto de "Stockfish-GPT-Tutor" de la comunidad de software libre.
- **Motor de Ajedrez:** [Stockfish](https://stockfishchess.org/) a través de la API pública de [Lichess](https://lichess.org).
- **Explicaciones de IA:** OpenAI GPT API.
- **Librerías de interfaz:** [Chess.js](https://github.com/jhlywa/chess.js) y [Chessboard.js](https://chessboardjs.com/).
